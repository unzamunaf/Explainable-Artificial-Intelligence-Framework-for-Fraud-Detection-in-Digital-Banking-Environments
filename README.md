# Explainable-Artificial-Intelligence-Framework-for-Fraud-Detection-in-Digital-Banking-Environments
An Explainable AI framework for credit card fraud detection combining XGBoost + LightGBM ensemble with SHAP &amp; LIME explanations. Achieves F1-Score of 0.9438 and AUC-ROC of 0.9942 with full GDPR compliance.
